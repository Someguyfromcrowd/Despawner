This plugin provides a single command, /despawn, which allows you to despawn mobs in a given radius.

Flags available:

-h: only despawn hostile mobs
-n: only despawn NPCs (villagers/golems)
-a: only despawn animals and neutral mobs (animals/bats)

Flags are optional.

Syntax:

/despawn [radius] [flag]
/despawn [radius] [x] [z] [flag]
/despawn [radius] [x] [z] [world] [flag]
